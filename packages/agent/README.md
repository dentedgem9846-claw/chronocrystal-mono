# @dentedgemclaw9846/chronocrystal-agent-core

This package contains the core functionalities for the ChronoCrystal agent.

## Installation

```bash
npm install @dentedgemclaw9846/chronocrystal-agent-core
```

## Usage

```typescript
import { initializeAgentCore } from '@dentedgemclaw9846/chronocrystal-agent-core';

console.log(initializeAgentCore());
// Output: "ChronoCrystal Agent Core initialized."
```
